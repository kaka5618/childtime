# 小程序发布材料草案

## 1. 小程序名称

推荐名称：

星星学习计划

推荐理由：

名字直接突出“学习计划”，家长一眼能理解用途；“星星”保留儿童感和奖励感，也不绑定某一套卡片主题。后续无论新增森林、海洋、恐龙、宇宙等卡包，都能放在“星星学习计划”这个产品框架里。

备选名称：

- 小星星计划本
- 成长星计划
- 小星计划
- 每日小计划
- 小小计划本

不建议使用：

- 星梦泡泡
- 星梦泡泡计划

原因：“星梦泡泡”更适合作为第一套卡包主题名，不适合作为整个小程序名。

## 2. 小程序简称

推荐：

星星学习计划

备选：

- 星星计划
- 小星学习计划
- 成长星

## 3. 小程序简介

推荐版本：

用每日计划、专注计时和卡片收集，陪孩子养成稳定的学习习惯。

更完整版本：

面向孩子的学习计划与习惯养成工具，通过每日任务、专注计时、能量收集和卡片奖励，帮助孩子一步步完成学习安排。

更温和版本：

把每日学习变成能量收集和卡片收藏，陪孩子认真完成自己的学习计划。

## 4. 服务类目建议

优先尝试：

- 教育 / 教育信息服务
- 工具 / 效率

选择建议：

如果后台允许当前主体选择“教育 / 教育信息服务”，可以优先选教育相关类目；如果教育类目要求资质或审核较严，可以选“工具 / 效率”，因为当前产品核心是学习计划、专注计时、习惯养成和虚拟收藏激励，不提供在线授课、直播课程、付费课程或教师服务。

注意：

最终类目以微信后台可选项和主体资质为准。提交前要确认页面功能与所选类目一致。

## 5. 图标方向

推荐图标概念：

一个可爱的黄色小星星抱着学习计划本，旁边有透明能量泡泡和小贴纸元素，背景使用浅天空蓝和柔和绿色，整体像儿童手帐贴纸。

已生成文件：

- 后台上传版：`release-assets/icons/star-study-plan-icon-144.png`
- 原始高清版：`release-assets/icons/star-study-plan-icon-source.png`

图标关键词：

- 黄色小星星
- 计划本
- 能量泡泡
- 铅笔或书本
- 云朵
- 柔和蓝绿
- 儿童贴纸感
- 学习计划

图标要求：

- 主体要大，缩小到微信列表里也能看清。
- 不要放文字，小图标里文字会糊。
- 不要用复杂背景，优先突出“小星星 + 学习计划本”。
- 建议导出 PNG，尺寸至少 1024 x 1024，文件小于 2MB。

## 6. 小程序头像说明

可填写/内部说明：

图标中的小星星和学习计划本代表孩子每天完成自己的学习安排；能量泡泡代表完成任务后的正向激励和卡片收集。

## 7. 隐私保护指引用途说明

微信身份标识 / OpenID：

用于识别同一微信账号，完成学习数据云端同步、云端恢复和账号状态管理。

孩子档案信息：

用于展示孩子昵称、头像贴纸、年龄段、每日学习目标和首页欢迎语。

学习任务信息：

用于记录任务名称、预计学习分钟数和完成状态，并计算能量收集进度。

收藏进度信息：

用于保存卡片收藏数量、最近开包记录、合成记录，并支持云端恢复。

不使用：

手机号、定位、通讯录、相册、摄像头、麦克风。

## 8. 审核备注

推荐填写：

本小程序为儿童学习计划和习惯养成工具，主要功能包括添加每日学习任务、专注计时、完成任务后获得虚拟卡片奖励、查看收藏进度。小程序不提供在线授课、直播课程、付费课程、交易服务或现实奖励兑换。

## 9. 截图素材建议

建议准备 3-5 张截图：

1. 首页：展示能量袋、今日计划、任务清单。
2. 专注页：展示倒计时和学习任务。
3. 开包页：展示宝箱开包动画或卡片结果。
4. 收藏本：展示卡片收藏网格。
5. 设置页：展示孩子档案、云同步和隐私说明。
