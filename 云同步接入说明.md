# 云同步接入说明

当前小程序端已经有“微信登录”和“同步云端”入口，并已打开云同步开关。完成下面配置和部署后即可同步。

## 1. 开通云开发

在微信开发者工具中开通云开发环境，并记录环境 ID。

## 2. 配置项目云函数目录

项目已在 `project.config.json` 顶层配置：

```json
{
  "cloudfunctionRoot": "cloudfunctions/"
}
```

## 3. 部署云函数

右键 `cloudfunctions/syncUserData`，选择“上传并部署：云端安装依赖”。

云函数会写入两个集合：

- `users`：孩子档案、当前主题、设置等轻量信息。
- `user_sync_snapshots`：任务、收藏、卡包状态的完整同步快照。

## 4. 云环境 ID

当前 `miniprogram/app.js` 的 `cloudEnvId` 留空，会使用开发者工具当前默认云环境。

如果你有多个云环境，建议填写指定环境 ID：

```js
cloudEnvId: '你的环境ID'
```

云同步开关当前为：

```js
cloudSyncEnabled: true
```

## 5. 验证

进入设置页，先点“微信登录”，再点“同步云端”。成功后设置页会显示最近同步时间。
