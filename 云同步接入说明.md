# 云同步接入说明

当前小程序端已经有“微信登录”和“同步云端”入口，默认不会上传数据。完成下面配置后，再打开同步开关。

## 1. 开通云开发

在微信开发者工具中开通云开发环境，并记录环境 ID。

## 2. 配置项目云函数目录

在 `project.config.json` 顶层加入：

```json
{
  "cloudfunctionRoot": "cloudfunctions/"
}
```

如果开发者工具已经自动改过 `project.config.json`，只添加这一项即可。

## 3. 部署云函数

右键 `cloudfunctions/syncUserData`，选择“上传并部署：云端安装依赖”。

云函数会写入两个集合：

- `users`：孩子档案、当前主题、设置等轻量信息。
- `user_sync_snapshots`：任务、收藏、卡包状态的完整同步快照。

## 4. 打开小程序端开关

在 `miniprogram/app.js` 中改为：

```js
cloudSyncEnabled: true
```

如需指定环境 ID，可在 `wx.cloud.init` 中加入：

```js
env: '你的环境ID'
```

## 5. 验证

进入设置页，先点“微信登录”，再点“同步云端”。成功后设置页会显示最近同步时间。
