# 星梦泡泡 · 卡片美术规范

> 版本：V1.0  
> 日期：2026-05-21  
> 用途：指导「星梦泡泡」主题卡池的卡面生成、人工筛选与前端展示  
> 适用范围：N / R / SR / SSR 四档稀有度卡片

---

## 一、主题定位

「星梦泡泡」是一套面向 6~12 岁儿童的原创泡泡精灵收藏卡。整体气质应当是柔软、梦幻、明亮、治愈、有收藏感，但不幼稚廉价。

核心视觉关键词：

- 圆滚滚
- 软乎乎
- 泡泡感
- 星星触角
- 糖果色
- 软胶 / 毛绒 / 果冻质感
- 潮玩盲盒风
- 梦境、星光、月光、彩虹、云朵

禁止方向：

- 不做人形角色
- 不出现真实人脸
- 不出现武器、战斗、攻击姿态
- 不恐怖、不阴暗、不尖牙利爪
- 不使用任何现成 IP 或近似知名角色
- 不在卡面生成可读文字、Logo、水印

---

## 二、统一角色基因

所有星梦泡泡角色必须共享以下基础特征：

| 项目 | 规则 |
|---|---|
| 身体 | 圆形或接近圆形，像泡泡、云团、果冻或软胶玩具 |
| 五官 | 点状眼睛或小星光眼，表情温和，不夸张 |
| 触角 | 至少 1 个星星形触角，SSR 可使用双星触角 |
| 四肢 | 可有短小手臂，不强调腿部，不做复杂动作 |
| 姿态 | 正面或 3/4 视角居中展示，友好、安静、轻微动态 |
| 材质 | 软胶、毛绒、果冻、透明泡泡等儿童友好材质 |
| 背景 | 与角色主题呼应，但不能抢主体 |

---

## 三、卡片通用规格

| 项目 | 规格 |
|---|---|
| 比例 | 3:4 竖向卡片 |
| 构图 | 全卡完整可见，角色居中，占卡面主视觉区域 |
| 边角 | 圆角卡片 |
| 文本 | 不生成可读文字 |
| 名牌区 | 底部保留空白名牌区域，后续由前端或模板叠加文字 |
| 稀有度区 | 右上角保留空白徽章区域，后续由前端或模板叠加 N/R/SR/SSR |
| 边框 | 各稀有度必须有清晰可识别的边框层级 |
| 缩略图 | 缩小后仍能区分稀有度和主体轮廓 |
| 外沿 | 卡片本体必须铺满画布，不允许出现白边、白色外框、截图式留白或卡片外背景 |

## 四、稀有度视觉层级

### 4.1 N · 普通卡

定位：基础可爱款，干净、柔和、低装饰。

代表样品：棉花泡泡。

| 项目 | 规则 |
|---|---|
| 边框 | 细白外框 + 浅粉或浅蓝内线 |
| 背景 | 浅色、干净、低复杂度 |
| 材质 | 毛绒、云朵、哑光软胶 |
| 装饰 | 少量星点或柔光，不使用明显闪卡效果 |
| 色彩 | 奶白、浅粉、浅蓝、浅桃、薄荷等低饱和色 |
| 高级感 | 保持精致，但不能像 SR/SSR 一样华丽 |

## 五、生产检查标准

每张图进入卡池前必须检查：

- 是否为 3:4 竖向卡片。
- 是否无可读文字、Logo、水印。
- 是否保留底部名牌区和右上徽章区。
- 是否没有白边、白色外框、截图式留白或卡片外背景。
- 角色是否符合“圆滚滚 + 星星触角 + 泡泡精灵”的统一基因。
- 边框是否清晰，缩略图状态下仍能辨认稀有度。
- 稀有度层级是否正确：N < R < SR < SSR。
- 是否适合 6~12 岁儿童，不恐怖、不战斗、不成人化。
- 是否与已有知名 IP 有明显相似风险。

## 六、四档样品定位

| 稀有度 | 样品名 | 用途 |
|---|---|---|
| N | 棉花泡泡 | 普通卡基础风格样板 |

| SR | 月光泡泡王子 | 银色、月光、稀有卡样板 |
